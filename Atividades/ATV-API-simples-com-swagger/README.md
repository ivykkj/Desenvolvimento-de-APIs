### Como Executar

- clone o repositório
- abra o terminal e instale as dependências necessárias(flask e flasgger)
    ```bash
    pip install Flask
    pip install flasgger
    ```
- inicie o servidor da API
    ```bash
    python app.py
    ```
- o servidor estará rodando e a API estará disponível em `http://127.0.0.1:5000/apidocs/`.
